Lynda Subs Download Script

script
 2017/09/22

Github Repository

Function:

This script can create download links of subtitles on Lynda.Com Page.

Usage:

  1. Install TamperMonkey or GreaseMonkey extension/addon to your Browser.
  2. Install the script via dashboard. (Create a new script).
  3. Login and refresh Lynda.Com course pages, check the dark sidebar in the lower left, you can then click and save the subtitles directly.
